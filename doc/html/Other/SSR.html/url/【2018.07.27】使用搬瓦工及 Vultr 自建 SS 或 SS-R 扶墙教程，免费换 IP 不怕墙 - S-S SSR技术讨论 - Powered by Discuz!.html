版块内帖子多为免费节点分享，而关于如何自建 SS 或 SS-R 节点的教程还很少，本次就详细讲一讲从购买 VPS 到搭建 SS 或 SS-R 节点的全过程。
目前 VPS 的价格并不高，选择自建节点后，不仅避免了频繁更换节点的麻烦，而且还保证了账户数据的安全。

第一部分、前言
我们一般途径下无法访问谷歌等网站是因为数据传回来时被 GFW 防火墙拦截了，而如果我们在国外 VPS 上搭建 SS 或 SS-R 节点，然后数据经过这台 VPS 的转发，我们就能够成功收到返回信息，即才能够成功浏览谷歌等网站。具体原理如下图所示。

第二部分、国外 VPS 选择
通过前言部分我们可以看到想要访问谷歌等网站，一台国外 VPS 是必不可少的。
目前售卖国外 VPS 的主机商有很多，楼主在用的只有搬瓦工和 VULTR 两个主机商，搬瓦工的 VPS 是楼主的主力扶墙机器，VULTR 则是拿来备用。在这里也推荐大家使用这两家，稳定不跑路，以下是两家的简单介绍。
1、搬瓦工主机商简单介绍
商家特点：价格低廉，服务稳定，速度较快，可一键切换机房，每 10 周可免费换一次 IP；
计费方式：按月份收费，当前最低年付 19.99 美元（平均每月十元钱）；
机房位置：有香港 PCCW、洛杉矶 CN2、洛杉矶 QNET、洛杉矶 MCOM 等机房。
2、VULTR 主机商简单介绍
商家特点：VPS 可以随时创建与删除，0.01 美元即可换 IP 一次；
计费方式：按小时收费，累计扣费一个月最低只要 2.5 美元；
机房位置：有洛杉矶、新加坡、东京、硅谷、伦敦等机房，全球共 15 个数据中心。
请根据需要选择一家，整体上来说，想要速度快的选择搬瓦工，怕 IP 被封的选择 VULTR。
